Downloading PIBASE.ligands
PIBASE.ligands is made available under the GPL license.

The installation guide describes how to install the PIBASE.ligands extension. Installation of the PIBASE.ligands database requires an existing PIBASE installation. To run your own mysql queries, all you need is the mysql dump below. The software package below is only necessary if you want to set up your own web server.
